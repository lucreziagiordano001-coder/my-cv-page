# Lucrezia Giordano - Personal Portfolio

Questo è il mio portfolio professionale da **Junior Full Stack Developer**. Il sito è progettato per presentare le mie competenze, il mio percorso formativo e i progetti che ho realizzato.

## 🚀 Live Demo
Puoi vedere il sito web completato qui: [Lucrezia Giordano Portfolio](https://lucreziagiordano001-coder.github.io/mycvpage/)

## 🛠️ Tecnologie Utilizzate
* **HTML5 / CSS3** (Semantica ed accessibilità)
* **Sass (SCSS)** (Strutturazione dello stile e gradienti personalizzati)
* **Bootstrap 5** (Layout responsive e componenti)
* **JavaScript** (Logica dinamica)
* **EmailJS** (Integrazione del form di contatto lato client)

## 📁 Struttura del Progetto
Il progetto segue una struttura pulita e modulare, dividendo gli asset (CSS, immagini, JS) in cartelle dedicate per garantire manutenibilità.
